Import invoice from Excel file / Import righe da Excel

This module add a button [import] in order to import sale order lines from a Excel file.

Excel file must have following structure:

NomeCodiceQuantita
Prodotto AlphaAAA10
Prodotto BetaBBB20

NomeCodiceQuantita
Prodotto AlphaAAA10
Prodotto BetaBBB20
