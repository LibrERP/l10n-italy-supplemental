Sort invoice lines by sale order, DdT, sequence, id / Rinumera righe fattura

This module give you to power of renumber a sale invoice line by:

  • sale order
  • DdT number
  • invoice sequence
  • line id

Rinumera righe fattura

Questo modulo permette di rinumerare le righe di una fattura per:

  • numero d'ordine
  • numero di DdT
  • sequenza della riga
  • ID della riga
